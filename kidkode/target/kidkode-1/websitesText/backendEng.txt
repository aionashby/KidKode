Back-End Engineering
A Back-end Engineer is responsible for creating the structure of the information used behind the scenes of an app or website. Back-end Engineers primarily write code to be used by front-end engineers and UX designers. Their day to day work consists of improving speed and stability, creating security structures, writing reusable code libraries and generating data storage solutions.
